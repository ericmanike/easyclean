# Easy Clean & Laundry Services

Professional cleaning and laundry services website. Reclaim your time with Easy Clean's professional cleaning and laundry pickup & delivery solutions.

## 🚀 Key Features
- **Responsive Design**: Optimized for mobile, tablet, and desktop viewing.
- **Interactive Gallery**: Before & after results slider.
- **Service Catalog**: Detailed information on residential, office, and specialized cleaning.
- **Booking System**: Streamlined form for quick service requests.
- **Modern UI**: Clean aesthetics with glassmorphism and smooth transitions.

## 🛠️ Project Structure
- `index.html`: Home page - overview of services and company values.
- `services.html`: Full list of service offerings.
- `gallery.html`: Portfolio of completed work with interactive comparison.
- `testimonials.html`: Client reviews and ratings.
- `contact.html`: Contact information, booking form, and FAQs.
- `styles.css`: Custom CSS variables and layout styling.
- `script.js`: Interactive elements and form handling.

## 💻 Technical Stack
- **HTML5**: Semantic structure.
- **CSS3**: Vanilla CSS with custom properties (variables) for the design system.
- **JavaScript**: ES6+ for interactive components.
- **Icons**: Font Awesome 6.5.1.
- **Fonts**: Inter (Google Fonts).

## 📂 Installation
1. Clone the repository.
2. Open `index.html` in any web browser.
